## Requirements
- [NativeUI](https://github.com/Guad/NativeUI/releases)
- [mysql-async](https://github.com/ESX-Brasil/mysql-async)

## -- Additional --
- [blip_info](https://github.com/glitchdetector/fivem-blip-info)


## Installation
- Add this in your `server.cfg`:

```
start mfp_banking
```
